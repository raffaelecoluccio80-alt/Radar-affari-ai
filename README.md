# Radar Affari AI

MVP per Telegram + GitHub + Railway.

Variabili Railway:
- TELEGRAM_BOT_TOKEN (obbligatoria)
- CHECK_MINUTES=15
- SOURCE_URLS=url1,url2
- KEYWORDS=hilti,festool,makita,bici elettrica,ebike,faema,rational

Comandi Telegram:
- /start
- /test
- /status
- /scan
- /stop

Nota: il progetto non supera login, CAPTCHA o protezioni dei siti.
